# Curso Profesional de Manicura y Pedicura

## 1. Descripción del Proyecto
Plataforma educativa online para aprendizaje de manicura y pedicura dirigida a estudiantes principiantes y avanzados. Ofrece módulos estructurados, práctica interactiva, recursos descargables, evaluaciones automáticas y una comunidad activa. La propuesta de valor es el aprendizaje completo y certificado desde cero hasta nivel profesional.

## 2. Estructura de Páginas
- `/` - Página principal (landing page única con todas las secciones)
  - `#introduccion` - Módulos introductorios
  - `#tecnicas` - Técnicas avanzadas
  - `#practica` - Práctica interactiva
  - `#recursos` - Recursos descargables
  - `#evaluaciones` - Evaluaciones automáticas
  - `#foro` - Foro de discusión

## 3. Características Principales
- [x] Navegación fija con scroll suave entre secciones
- [x] Hero con imagen de fondo y llamado a la acción
- [x] Módulos introductorios con descripción de contenido
- [x] Sección de técnicas avanzadas con galería de técnicas
- [x] Práctica interactiva paso a paso
- [x] Biblioteca de recursos descargables (PDFs, videos, guías)
- [x] Evaluaciones automáticas con puntuación inmediata
- [x] Foro de discusión con categorías y formulario de preguntas
- [x] Footer con formulario de suscripción

## 4. Datos / Modelo
- Todos los datos son mock estáticos (sin base de datos)
- Formularios: newsletter y envío de preguntas al foro via Readdy Forms

## 5. Integraciones de Backend / Terceros
- Supabase: No requerido (datos estáticos)
- Formularios: Readdy Forms para newsletter y foro
- Shopify: No requerido
- Stripe: No requerido
